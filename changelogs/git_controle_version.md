Changelogs/releases é uma ferramenta destinada à comunidade MIBNA para controle de versões usado principalmente no desenvolvimento de software, mas pode ser usado para registrar o histórico de edições de qualquer tipo de arquivo.

Gerencie repositórios git com com vários níveis de controle de acesso para manter seu código com segurança.

Mantenha histórico de revisões de código e melhore a colaboração com merge requests.
